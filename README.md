## Instruções dos Comandos GIT - Realeses
Segue abaixo as Intruções de Criação de Realeses "Tags" e Push:

Tag 0.1.0
```sh
$ git tag 0.1.0
```
... Commits, Correções de Erros...

Tag 0.1.1
```sh
$ git tag 0.1.1
```

Push para o Repositório Remoto do GitHub:
```sh
$ git push origin master
$ git push origin master --tags
```
